The attached CV Assessment project is a full-stack web application developed using the MERN stack (MongoDB, Express.js, React.js, and Node.js) that intelligently evaluates resumes using Natural Language Processing techniques. A key component of the system is the use of TF-IDF (Term Frequency–Inverse Document Frequency), which helps in analyzing and ranking CVs based on their relevance to a given job description. The system processes textual data from resumes, converts it into numerical vectors using TF-IDF, and then compares these vectors with job requirements to identify the most suitable candidates. This approach ensures that important keywords and skills are given appropriate weight, improving the accuracy of candidate matching. The platform also integrates modern UI frameworks like Tailwind CSS and Material UI to provide an interactive user experience, making the assessment process efficient, automated, and scalable for recruitment purposes.
